### 🛒 Martivo – MERN E-Commerce Platform

A full-stack e-commerce web application built with the **MERN stack**.

### Key Features

• Product listing and product details
• Category filtering
• Shopping cart system
• Checkout process
• Order creation
• Order success page
• Order history for users

React • Node.js • Express.js • MongoDB • REST APIs • Context API
